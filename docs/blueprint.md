# **App Name**: SugarCrush Bakers

## Core Features:

- Interactive Menu: Display an interactive menu with cake, cupcake, brownie, and other product listings, each featuring a hover-to-reveal 'Order Now' button. Utilizes a card grid with smooth animations.
- Animated Hero Section: Implement a full-screen hero section with a pastel gradient background, animated cake illustrations or floating elements, and a gentle entrance animation for headlines and subtext using Framer Motion.
- Online Ordering Tool: Provide access to a form for customers to specify order details. An LLM analyzes the customer request, provides recommendations for items or flavors to add, or requests clarification, as appropriate. After submission, a 'thank you' animation is displayed, enhancing user engagement.
- Testimonial Carousel: Include a testimonials section displaying customer reviews. The carousel will horizontally slide through testimonials with quote icons, pastel backgrounds, star ratings, and profile images.
- Contact Section with Google Maps: Embed a Google Maps display to show the bakery's location. This section also includes contact information such as phone number, email, and Instagram handle, accompanied by animated social icons.
- Floating WhatsApp Chat Button: Implement a floating WhatsApp chat button on the bottom-right corner for easy customer communication.

## Style Guidelines:

- Primary color: Pastel pink (#FFD6E0) for a sweet, inviting feel.
- Background color: Creamy beige (#FFF7F0) to provide a warm, comforting backdrop.
- Accent color: Warm brown (#4B2E2B) for text and interactive elements, providing contrast and grounding the palette.
- Headings: 'Playfair' (serif) for an elegant look. Note: currently only Google Fonts are supported.
- Body: 'Poppins' (sans-serif) for clean, rounded readability. Note: currently only Google Fonts are supported.
- Use cupcake and heart icons for a playful, warm feel.
- Employ rounded card corners, smooth shadows, and subtle gradients for a modern look.
- Incorporate soft entrance animations, hover lift effects, and fade-in scroll animations for a delightful user experience.
